Website Evaluation

W3 Schools

W3 Schools is a website that helps teach people website development focused coding.

  • Contrast - W3's use of contrast is very well done. Certain sections of the website utilize a darker background and lighter text to ensure visibility.
  • Repitition - There is a great sense of unity on the webpage. Each language, explaination and color use is repetitive.
  • Alignment - Everything is aligned very well. Each section is very similar to the element above it.
  • Proximity - W3 Schools uses proximity well, but it separates sections using colors very well.

Land Rover Forums

Land Rover Forums is a forum for people that really like first generation Land Rovers.

  • Contrast - Land Rover Forums does not utilize contrast very well. They do use black on white often, but text is small and blends together.
  • Repitition - The color scheme is repeated well, but not much else is. There are several elements that appear once.
  • Alignment - The alignment is not terrible, but there are two containers right next to one another and it becomes a bit too much to look at.
  • Proximity - Similar items are matched together quite well.
